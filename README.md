#技术路线1
1. 调取百度地图API并下载为png格式（中心点为GPS读取点，去除显示的字与其他图标）
2. 通过图像处理的方式，将png格式转为二维占用栅格地图
3. 将自身的GPS坐标转化为二维栅格地图坐标，通过AMSL实时定位确定方向（或者是陀螺仪和罗盘）
4. 接收目标GPS坐标并转化为二维栅格地图坐标
5. 导入并配置自主导航功能包
